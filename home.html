SYSTEM ARCHITECTURE

SightX operates as a full-stack web application powered by a Flask server that bridges the AI engine to your browser in real-time.

The YOLO detection engine processes raw camera frames, while ByteTrack maintains persistent object tracking across frames for smooth, uninterrupted surveillance.

Telemetry data flows via REST API endpoints, feeding the dashboard's live gauges, threat meters, and incident logs — all without page reloads.
